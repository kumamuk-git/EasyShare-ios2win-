①PCのIPアドレスの固定
　1.1：windows左下の検索窓に「cmd」と入力してコマンドプロンプトを開く
　1.2：「ipconfig -all」と入力し、エンター。ipアドレス関連の情報が出てくる。
　1.3：https://www.buffalo.jp/support/faq/detail/15257.html 
	上記を参考にipアドレスを手動で設定する。1.3で確認したものを入力。

②node.jsのインストール
　2.1：https://medium-company.com/node-js%E3%81%AE%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%AB%E6%89%8B%E9%A0%86/
	ここ参考にインストール

③server.jsの配置＆モジュールのインストール
　3.1：お好きな場所にserver.jsを配置。
　3.2：server.jsが配置されたフォルダをエクスプローラで開いた状態で、エクスプローラ上部ののパスが書かれた部分
	（ドキュメントのAというフォルダをひらいていたら　（フォルダのマーク）＞ドキュメント＞A　と表示されている部分）
	をクリックし、「cmd」と入力しエンター。
	するとそのディレクトリでコマンドプロンプトが起動される。
　3.3：「npm install express open」を入力し、エンター。モジュールがインストールされる。
	node_modulesというフォルダができていればOK。

④iPhoneショートカットの設定。
　4.0：https://www.icloud.com/shortcuts/f9d65a724bb14ed08ed655ff13f70081　<-ショートカットリンク
　4.1：ショートカットアップリを開き、共有したショートカットの編集画面へ。
　4.2：「http://(ipアドレス):8080/の内容を取得」のipアドレスの部分を①で設定したものへ変更。

⑤node.jsのプロセスマネージャ（pm2）のインストール
　5.1：3.2と同様の方法でコマンドプロンプトを起動。
　5.2：「npm install -g pm2」でインストール。
　5.3：「npm install -g pm2-windows-startup」も実行。

⑥検証
　6.1：3.2と同様の方法でコマンドプロンプトを起動。
　6.2：「pm2 start server.js」と入力し、エンター。きっと無事サーバが立つ。
　6.3：iphoneでsafariを開き、共有ボタンからショートカットを実行。
　6.4：きっとwebサイトがwindowsで開かれる。
　6.5：（コマンドプロンプトを閉じないで！）

⑦windows起動と同時に起動されるよう設定
　7.1：「pm2 save」を入力し、エンター。
　7.2：「pm2-startup install」を入力し、エンター。これでwin起動時に勝手にサーバ立ててくれます。
